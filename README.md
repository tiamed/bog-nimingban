## 功能
- 板块
- 收藏
- 浏览历史
- 发言历史
- 获取/添加饼干
- 串内跳页
- 串分享
- 串跳转
- 只看发布者
- 图片保存
- 图片分享
- 加载语录
- 定制显示：字体大小、行距、最大行数、夜间模式、图片位置、图片缩放、串内图片数...
